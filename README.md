## Description 🍉
discord bot that implements a watermelon farm and replies to some trigger message prefixes

## Install packages 🍉
```
sudo dnf install cmake g++ 
libssl-dev zlib1g-dev 
pkg-config libpqxx-dev
```

## Start bot 🍉
```
./build.sh
./build/krash-bot
```
